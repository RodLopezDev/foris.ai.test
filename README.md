# General

Version de Python

```
Python 3.11.4
```

# Run

Archivo como input de ejecución

```
cat data.txt | python main.py
```

Cargar el nombre del archivo como parámetro

```
python main.py data.txt
```

# Tests

Instalar pytest

```
pip install pytest
```

Correr tester

```
pytest
```
